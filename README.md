## 📝 Example Profile Configuration

Create a file named `torch_profile.yaml` in the app root directory:

```yaml
model_preference: dynamic
primary_api:
  provider: openai
  model: gpt-4-turbo
  temperature: 0.7
secondary_api:
  provider: claude
  model: claude-3-5-sonnet-20241022
  temperature: 0.8
fallback_strategy: chain
ui_language: auto
transcription_lang: ["en", "es", "fr", "de", "zh"]
local_model_path: ./models/llama-3-8b.Q4_K_M.gguf
patch_verification: auto_disable
```
